## Install

1. Following the instructions on Lean Official Website and install the Lean tool chain. VSCode strongly recommended.

2. Clone this repository.

3. Open `Tutorials/Peano.lean`, and you should be able to see the Lean InfoView as well. Let `lake` compile the dependencies, which takes several minutes. You can press `Restart File` to retry the compilation.

4. After compilation, you should see `No info found.` in the Lean InfoView. This means the Lean environment is set up.
